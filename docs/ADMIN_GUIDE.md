## ğŸ”„ How Availability Updates Work

### When sessions are completed:
1. Staff marks session as "Complete" in admin dashboard
2. Time slot **automatically becomes available** again
3. New clients can book that time

### When dates are blocked:
1. Staff blocks a date in admin dashboard
2. Date becomes **unavailable immediately**
3. Clients see "counselor is not available on this date"

### When dates are unblocked:
1. Staff unblocks the date
2. Date becomes **available immediately**
3. Clients can book available time slots
